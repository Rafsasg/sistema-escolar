    Acerca del Sistema

    Este sistema escolar está diseñado para facilitar la gestión de estudiantes, notas, informes y matrículas. Con un enfoque mobile-first, se busca brindar una experiencia sencilla, rápida y agradable.

    Desarrollado con tecnologías modernas como HTML5, CSS3 y JavaScript, e integrado con Firebase para el manejo en tiempo real de datos.
